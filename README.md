# 📦 Elite Courier Pro [Build 1.1.0]

**Elite Courier Pro** is a high-performance, enterprise-grade Logistics Management Solution developed with **Python**. Designed for modern desktop environments, it features a scalable architecture, dynamic UI engine, and robust data management capabilities.

## 🚀 Architectural Highlights

- **📐 Adaptive UI Scaling:** Features a custom-built zoom engine (`0.8x` to `1.5x`) that dynamically resizes fonts, widgets, and data grids without compromising layout integrity.
- **🎨 Multi-Theme Engine:** Implements a professional theme-switching logic with high-contrast **AMOLED Dark** and **Emerald Light** color palettes.
- **🏗️ Object-Oriented Design:** Developed using strict **OOP principles** to ensure code maintainability and long-term scalability.
- **🖥️ High-DPI Support:** Integrated Windows OS DPI-awareness to ensure crystal-clear visuals on 4K and Retina displays.

---

## 🛠️ Core Functionalities

- **Shipment Lifecycle Tracking:** Real-time status updates including `In Transit`, `Delivered`, `Returned`, and `Voided`.
- **Financial Audit System:** Automated revenue aggregation and billing status reporting.
- **Smart Data Grid:** A responsive `Treeview` implementation with dynamic column scaling and multi-select capabilities.
